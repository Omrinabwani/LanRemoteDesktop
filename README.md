A LAN-based remote desktop application that allows one computer to view and control another computer over a local network.
The system is built using C# and WPF and communicates through a custom TCP protocol designed specifically for remote screen streaming and input control.
This project demonstrates concepts in: networking, client–server architecture, custom binary protocols, screen capture and compression, remote input handling, Features, Remote screen viewing over LAN 
Mouse control of the host machine, Keyboard input forwarding, JPEG-based screen streaming, Custom TCP message protocol, Real-time display in a WPF client.
